The CTC100 Cryogenic Temperature Controller is a high performance instrument that can monitor and control temperatures with millikelvin resolution.

### Sensor Inputs

The CTC100 offers four temperature inputs that can read RTDs, thermistors, and diodes. Each temperature input channel has its own 24-bit ADC with eleven input ranges, and is equipped with its own independent excitation current source. Standard calibration curves are included for various sensors, with support for custom calibration curves containing up to 400 points each.

### Powered and Analog Voltage Outputs

The controller provides two heater outputs delivering up to 100 W of power each to 25 Ω heaters. Additionally, four analog voltage I/O channels enable external amplifier-driven heater control.

### PID Feedback Control

Up to six feedback control loops are available for precise temperature management. Any of the four channels can serve as input to each feedback loop, with adjustable time constants ranging from 200 milliseconds to 10 hours. Up to ten PID parameter sets can be stored per channel, and setpoints support fixed-rate ramping.

### Analog and Digital I/Os

Four general-purpose ±10 V voltage I/O channels with 24-bit ADC conversion are provided alongside eight optoisolated digital I/O lines. Four 5 A relays facilitate process control, while three virtual channels allow display and logging of calculated values.

### Numeric and Graphical Display

The instrument features a color LCD touchscreen supporting mixed numeric and graphical presentations. Up to eight channels can plot on single or independent-axis graphs.

### Data Logging

The CTC100 stores up to 1 million readings per channel internally with USB export capability for extended storage.

### Computer Communications

Remote control occurs via USB, Ethernet, RS-232, or optional GPIB interfaces using text commands corresponding to front-panel controls. Macro programs can be uploaded via communications ports or USB storage devices.
